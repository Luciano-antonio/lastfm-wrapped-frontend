# Last.fm Wrapped - Frontend

## O que é
Interface do Last.fm Wrapped, onde usuários podem visualizar suas estatísticas musicais em tempo real, incluindo top artistas, músicas e histórico recente.

## Preview

<img width="1920" height="1037" alt="image" src="https://github.com/user-attachments/assets/c888904f-c11d-48b7-a706-fd346a184c33" />

## Features
- Autenticação com OAuth do Last.fm
- Top 50 músicas mais ouvidas
- Top 50 artistas mais ouvidos
- Últimas 50 músicas ouvidas
- Atualização automática a cada 30 segundos na seção "Tocadas recentemente"

## Tecnologias
- React / Vite / TypeScript
- Tailwind CSS
